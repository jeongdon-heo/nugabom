# 🌱 누가봄 — 교사용 사용 매뉴얼

**초등학교 생활기록부 누가기록 도우미**

버전 1.0 | 2026년 3월  
https://nugabom.vercel.app

---

## 1. 누가봄이란?

누가봄은 초등학교 담임교사의 생활기록부 작성 부담을 줄여주는 웹 도구입니다.

수업 중 관찰한 학생의 모습을 스마트폰이나 PC에서 간편하게 기록하고, AI가 생활기록부 초안을 자동으로 생성해 줍니다.

### 주요 특징

| 기능 | 설명 |
|------|------|
| 📱 모바일 입력 | PWA 지원 — 스마트폰 홈 화면에 추가하여 앱처럼 사용 |
| 📚 성취기준 기반 | 2022 개정 교육과정 1~6학년 전 교과 성취기준 내장 |
| 🤖 AI 초안 생성 | Gemini 2.0 Flash로 창체·교과·행동특성 초안 자동 작성 |
| 📋 나이스 정리 | 영역별 정리 + 복사 + CSV 내보내기로 나이스 입력 간편화 |
| 💾 로컬 저장 | IndexedDB에 저장 — 서버 전송 없이 브라우저에서만 관리 |

## 2. 시작하기

### 2-1. 접속 방법

브라우저에서 **https://nugabom.vercel.app** 에 접속합니다.

크롬, 사파리, 엣지 등 최신 브라우저에서 모두 사용할 수 있습니다.

### 2-2. 스마트폰에 앱으로 설치 (PWA)

**아이폰 (Safari)**

① Safari에서 누가봄 주소 접속  
② 하단 공유 버튼(□↑) 탭  
③ "홈 화면에 추가" 선택  
④ 홈 화면에 누가봄 아이콘이 생성됩니다

**안드로이드 (Chrome)**

① Chrome에서 누가봄 주소 접속  
② 상단 "홈 화면에 추가" 배너 또는 메뉴(⋮) → "앱 설치"  
③ 홈 화면에서 앱처럼 실행됩니다

> 💡 **TIP** 설치하면 인터넷 연결 없이도 기록 입력이 가능합니다 (AI 기능은 인터넷 필요).

### 2-3. Gemini API 키 설정

AI 기능(관찰 문구 자동 생성, 초안 생성)을 사용하려면 Google Gemini API 키가 필요합니다.

**API 키 발급 방법**

① https://aistudio.google.com/apikey 접속  
② Google 계정으로 로그인  
③ "API 키 만들기" 클릭  
④ 생성된 키(AIzaSy...)를 복사

**앱에 키 등록**

① 누가봄 하단 ⚙️ 설정 탭 이동  
② "🔑 Gemini API 키" 항목에 복사한 키를 붙여넣기  
③ 키는 브라우저에만 저장되며 외부로 전송되지 않습니다

> ⚠️ **주의** API 키는 개인용입니다. 다른 사람과 공유하지 마세요. Gemini 2.0 Flash는 무료 할당량이 넉넉하여 일반적인 사용에는 비용이 발생하지 않습니다.

## 3. 초기 설정

### 3-1. 학급 정보 설정

① ⚙️ 설정 탭으로 이동합니다.  
② "🏫 학급 정보" 옆의 [수정] 버튼을 누릅니다.  
③ 학년(1~6)과 반을 선택/입력하고 [저장]을 누릅니다.  
④ 학년을 변경하면 해당 학년의 교과와 성취기준이 자동으로 전환됩니다.

### 3-2. 학생 명단 설정

기본으로 25명의 샘플 이름이 입력되어 있습니다. 실제 학생 이름으로 수정하세요.

① ⚙️ 설정 탭 → "👥 학생 명단" 옆의 [수정] 버튼  
② 각 학생의 이름과 성별을 수정합니다.  
③ 학생 추가: 하단 [+ 학생 추가] 버튼  
④ 학생 삭제: 해당 학생 옆의 ✕ 버튼  
⑤ 완료 후 [저장] 버튼을 누릅니다.

> ⚠️ **주의** 학생 삭제 시 해당 학생의 누가기록도 접근이 어려워질 수 있으니 신중하게 진행하세요.

## 4. 누가기록 입력

✏️ 기록 탭에서 학생의 수업 관찰 내용을 입력합니다. 두 가지 입력 모드를 제공합니다.

### 4-1. 📚 교과 모드 (성취기준 기반)

교과 수업 관찰에 최적화된 입력 방식입니다.

**입력 흐름**

| 단계 | 화면 | 설명 |
|------|------|------|
| ① | 학생 선택 | 상단 학생 스크롤에서 기록할 학생을 탭합니다. 번호 뱃지로 기존 기록 건수를 확인할 수 있습니다. |
| ② | 교과 선택 | 국어, 수학, 사회, 과학 등 교과 버튼을 탭합니다. (⌛ 표시는 성취기준 미등록 교과) |
| ③ | 단원 선택 | 해당 교과의 단원 목록이 표시됩니다. 수업 중인 단원을 선택합니다. |
| ④ | 성취기준 선택 | 단원 내 성취기준이 표시됩니다. 관찰할 성취기준을 선택하면 AI가 관찰 문구를 자동 생성합니다. |
| ⑤ | 문구 선택 | AI가 생성한 문구 중 해당 학생에게 맞는 것을 탭하여 선택합니다. 여러 개 선택 가능. |
| ⑥ | 저장 | [기록 저장] 버튼을 탭하면 성취기준 코드가 자동으로 태깅되어 저장됩니다. |

> 💡 **TIP** 관찰 문구는 상(🟢 우수) / 중(🔵 보통) / 하(🟠 노력 필요) 수준별로 분류되어 있어 수준별 기록이 편리합니다.

> 💡 **TIP** [🔄 다시 생성] 버튼으로 새로운 문구를 받을 수 있고, 추가 메모 입력란에 직접 내용을 더할 수도 있습니다.

### 4-2. ✏️ 자유 모드

교과 외 관찰이나 빠른 메모에 적합한 카톡형 입력 방식입니다.

**해시태그 자동 분류**

메모에 해시태그를 붙이면 영역이 자동으로 분류됩니다.

| 해시태그 | 영역 | 입력 예시 |
|---------|------|----------|
| #국어, #수학, #과학... | 교과 | #수학 분수의 덧셈을 정확하게 계산함 |
| #자율, #동아리, #진로 | 창체 | #자율 학급 회의에서 적극적으로 의견 제시 |
| #협동, #리더십, #발표 | 행동특성 | #협동 모둠 활동에서 친구를 배려하며 역할 수행 |
| #배려, #노력, #태도 | 행동특성 | #성장 발표에 대한 자신감이 눈에 띄게 향상됨 |

**전체 해시태그 목록**

| 해시태그 | 영역 | 세부 카테고리 |
|---------|------|-------------|
| #국어, #수학, #사회, #과학, #영어, #도덕, #체육, #음악, #미술, #실과 | 교과 | 각 교과명 |
| #바른생활, #슬기로운생활, #즐거운생활 | 교과 | 각 교과명 (1~2학년) |
| #자율, #자치 | 창체 | 자율자치 |
| #동아리 | 창체 | 동아리 |
| #진로 | 창체 | 진로 |
| #협동 | 행동특성 | 교우관계 |
| #리더십 | 행동특성 | 리더십 |
| #발표 | 행동특성 | 학습태도 |
| #배려 | 행동특성 | 교우관계 |
| #노력 | 행동특성 | 학습태도 |
| #태도 | 행동특성 | 학습태도 |
| #성장 | 행동특성 | 기타 |
| #교우, #교우관계 | 행동특성 | 교우관계 |
| #생활 | 행동특성 | 생활습관 |
| #정서 | 행동특성 | 정서 |

**템플릿 활용**

📋 버튼을 탭하면 자주 쓰는 관찰 키워드 템플릿이 표시됩니다.

- 📚 학습: 적극 발표, 또래 도움, 과제 성실, 심화 탐구, 창의적 표현 등
- 🤝 사회성: 갈등 조절, 양보 배려, 모둠 주도, 경청 존중, 규칙 준수 등
- 🌱 성장: 자신감 향상, 참여도 개선, 꾸준한 노력, 태도 변화 등

> 💡 **TIP** 템플릿을 선택하면 입력창에 자동으로 키워드가 삽입됩니다. 뒤에 구체적인 관찰 내용을 이어서 작성하세요.

## 5. 대시보드

📊 대시보드 탭에서 학급 전체의 기록 현황을 한눈에 파악할 수 있습니다.

### 5-1. 요약 카드

총 기록 건수, 학생당 평균 건수, 기록 부족 학생 수가 상단에 표시됩니다.

### 5-2. 기록 현황 매트릭스

학생별로 창체·교과·행동특성 영역의 기록 건수가 표 형태로 표시됩니다.

| 상태 | 의미 |
|------|------|
| 🟢 충분 | 6건 이상 — 생활기록부 작성에 충분한 기록량 |
| 🟡 보통 | 3~5건 — 추가 기록이 있으면 좋겠음 |
| 🔴 부족 | 0~2건 — 기록 보충 필요 |

필터 버튼(전체/부족/보통/충분)으로 특정 상태의 학생만 볼 수 있습니다.

### 5-3. 학생 상세 보기

매트릭스에서 학생 이름을 탭하면 해당 학생의 전체 기록을 영역별로 확인할 수 있고, 기록 수정·삭제도 가능합니다.

## 6. 나이스 정리

📋 정리 탭에서 학생별 누가기록을 나이스 입력 형식으로 정리하여 볼 수 있습니다.

### 6-1. 사용 방법

① 학생 선택 (드롭다운 또는 ◁▷ 버튼으로 이전/다음 학생 이동)  
② 영역 탭 선택: 창체 / 교과 / 행동특성  
③ 영역별로 세부 카테고리(교과명, 창체 영역 등)로 그룹핑되어 날짜순 정렬됩니다.  
④ [📋 전체 복사] 버튼으로 해당 영역 전체 텍스트를 클립보드에 복사  
⑤ 각 카테고리별 [복사] 버튼으로 부분 복사도 가능

### 6-2. 나이스 입력용 텍스트

하단에 나이스에 바로 붙여넣을 수 있는 형태의 텍스트가 표시됩니다.

교과는 과목별, 창체는 영역별로 구분되어 있어 나이스 항목에 맞게 복사-붙여넣기하면 됩니다.

### 6-3. 전체 CSV 내보내기

상단 [📊 전체 학생 CSV 내보내기] 버튼을 누르면 전체 학생의 누가기록이 엑셀에서 열 수 있는 CSV 파일로 다운로드됩니다.

CSV 항목: 번호, 이름, 영역, 세부영역, 날짜, 성취기준, 메모

## 7. AI 초안 생성

🤖 AI초안 탭에서 누가기록을 바탕으로 생활기록부 초안을 자동 생성합니다.

> ⚠️ **주의** 반드시 설정에서 Gemini API 키를 먼저 등록해야 AI 기능이 작동합니다.

### 7-1. 개별 생성

① 👤 개별 생성 모드 선택  
② 학생 선택 (누가기록이 있는 학생만 생성 가능)  
③ 생성 항목 선택: 창체 / 교과 / 행동특성 (복수 선택 가능)  
④ 분량(간결/보통/상세)과 어조(객관적/따뜻한) 설정  
⑤ [🤖 AI 초안 생성] 버튼 클릭  
⑥ 생성된 초안을 확인하고, [수정] 버튼으로 직접 편집할 수 있습니다.  
⑦ [복사] 버튼으로 영역별 또는 전체를 클립보드에 복사합니다.

### 7-2. 일괄 생성

① 👥 일괄 생성 모드 선택  
② 누가기록이 있는 학생 전체를 대상으로 순차 생성됩니다.  
③ 진행 바로 현재 진행 상황을 실시간 확인할 수 있습니다.  
④ 완료 후 학생별 접기/펼치기로 결과를 확인합니다.  
⑤ [📋 전체 복사] 또는 [📊 CSV] 버튼으로 내보낼 수 있습니다.

### 7-3. 기재 금지사항 자동 검증

AI가 생성한 초안에 다음 항목이 포함되면 자동으로 경고(⚠️)가 표시됩니다.

| 검사 항목 | 예시 |
|----------|------|
| 학교명 기재 | ○○초등학교, △△학교 등 구체적 학교명 |
| 사교육 관련 | 학원, 과외, 사교육, 학습지 등 |
| 부모 사회·경제적 지위 | 아버지 직업, 어머니 직업, 부모 직위 등 |
| 교외 수상 실적 | 교외 대회, 외부 수상, 전국 대회 등 |
| 과장 표현 | 천재, 최고의, 가장 뛰어난, 압도적, 완벽한 등 |

## 8. 데이터 관리

### 8-1. 데이터 저장 방식

누가봄의 모든 데이터(학급 정보, 학생 명단, 누가기록, API 키)는 브라우저의 IndexedDB에 저장됩니다.

서버에 전송되거나 저장되지 않으므로 학생 개인정보 보호에 안전합니다.

> ⚠️ **주의** 브라우저 데이터를 삭제하면 모든 기록이 함께 삭제됩니다. 정기적으로 백업(내보내기)하는 것을 권장합니다.

### 8-2. 내보내기 (백업)

① ⚙️ 설정 탭 → "💾 데이터 관리" → [📥 내보내기] 클릭  
② JSON 파일이 다운로드됩니다 (예: 누가봄_4-2반_2026-03-29.json)  
③ 이 파일을 안전한 곳(구글 드라이브 등)에 보관하세요.

> 💡 **TIP** 학기 중 매주 금요일에 내보내기를 하면 안심입니다.

### 8-3. 가져오기 (복원)

① ⚙️ 설정 탭 → [📤 가져오기] 클릭  
② 이전에 내보낸 JSON 파일을 선택합니다.  
③ 현재 데이터가 파일 내용으로 교체됩니다.

> ⚠️ **주의** 가져오기를 하면 현재 데이터가 모두 덮어씌워집니다. 필요하면 먼저 현재 데이터를 내보내기 하세요.

### 8-4. 기록 초기화

새 학기에 누가기록만 삭제하고 학급·학생 정보는 유지하고 싶을 때 사용합니다.

① ⚙️ 설정 탭 → [🗑 기록 초기화] 클릭  
② 확인 후 모든 누가기록이 삭제됩니다 (학급·학생 정보는 유지).

## 10. 추천 활용법

### 📅 일과 중 빠른 기록

수업 중 관찰한 내용을 바로 스마트폰에서 자유 모드로 입력합니다.

- 예: `#수학 김민준 분수의 덧셈을 정확하게 계산하고 친구에게 풀이를 설명함`
- 예: `#협동 이서연 모둠 활동에서 역할을 분담하고 친구들을 격려함`

### 📚 단원 마무리 후 교과 기록

단원이 끝나면 교과 모드에서 성취기준을 선택하고 AI 문구를 활용하여 체계적으로 기록합니다.

성취기준 코드가 자동 태깅되므로 나중에 교과학습발달상황 작성 시 편리합니다.

### 📊 주간 점검

매주 금요일에 대시보드에서 기록 현황을 확인합니다.

부족(🔴) 학생이 있으면 다음 주에 해당 학생을 집중 관찰합니다.

내보내기로 주간 백업도 함께 합니다.

### 🤖 학기말 초안 생성

학기말에 AI초안 탭에서 일괄 생성으로 전체 학생의 생활기록부 초안을 한 번에 만듭니다.

생성된 초안을 검토·수정한 후 나이스에 입력합니다.
